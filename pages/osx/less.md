# less

- Opens a file for reading
- Allows movement and search
- Doesn't read the entire file (suitable for logs)

## Open a file

`less source_file`

## Page up / down

`d (next), D (previous)`

## Start / end of file

`g (start), G (end)`

## Search for a string

`/something    then   n (next), N (previous)`

## Exit

`q`
